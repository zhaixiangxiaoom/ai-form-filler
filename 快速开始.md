# AI Form Filler - 快速开始指南

## 📦 安装步骤

### 获取项目代码

**方式一：从 GitHub 克隆**
```bash
git clone https://github.com/zhaixiangxiaoom/ai-form-filler.git
cd ai-form-filler
```

**方式二：下载 ZIP**
- 访问 GitHub 仓库页面
- 点击 **Code** → **Download ZIP**
- 解压下载的文件

### 第一步：加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的 **开发者模式** 开关
4. 点击 **加载已解压的扩展程序** 按钮
5. 选择您下载或克隆的 `ai-form-filler` 文件夹
6. 扩展图标将出现在浏览器工具栏中

### 第二步：获取阿里云百炼 API 密钥

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册或登录您的账户
3. 进入 API-KEY 管理页面
4. 点击 **创建新的 API-KEY**
5. 复制生成的密钥（只显示一次！）

### 第三步：配置扩展

1. 点击浏览器工具栏中的 AI Form Filler 图标
2. 切换到 **Settings** 标签页
3. 粘贴您的阿里云百炼 API 密钥
4. （可选）配置其他选项：
   - **API URL**: 保持默认（使用百炼官方 API：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`）
   - **AI Model**: 
     - Qwen-Turbo - 速度快，成本低
     - Qwen-Plus - 平衡性能和质量（推荐）
     - Qwen-Max - 质量最好，成本较高
   - **Language**: 选择生成数据的语言（中文/英文等）
   - **Auto-fill**: 是否自动填充检测到的表单
5. 点击 **💾 Save Settings** 保存设置

## 🚀 使用方法

### 基本使用流程

1. **打开网页**：导航到包含表单的网页
2. **打开扩展**：点击工具栏中的 AI Form Filler 图标
3. **检测表单**：点击 **🔍 Detect Forms** 按钮
4. **查看预览**：在 Field Preview 中查看检测到的字段
5. **生成并填充**：点击 **✨ Generate & Fill** 按钮
6. **完成！**：表单将自动填充 AI 生成的真实数据

### 支持的表单字段类型

✅ 文本输入框（text、email、phone、url 等）
✅ 多行文本框（textarea）
✅ 下拉选择框（select）
✅ 单选按钮（radio）
✅ 复选框（checkbox）
✅ 数字输入框（number）
✅ 日期输入框（date）
✅ **富文本编辑器（Rich Text Editors）**：
   - TinyMCE
   - CKEditor（4 和 5）
   - Quill
   - Froala
   - Summernote
   - contenteditable 元素
   - iframe 编辑器

## 💡 使用示例

### 示例 1：填写注册表单

1. 打开一个注册页面
2. 点击扩展图标 → Detect Forms
3. 查看检测到的字段（用户名、邮箱、密码等）
4. 点击 Generate & Fill
5. AI 将生成：
   - 真实的用户名
   - 有效的邮箱地址
   - 强密码
   - 其他必填信息

### 示例 2：填写中文表单

1. 在 Settings 中将 Language 设置为 Chinese
2. 打开需要填写的中文表单
3. 点击 Detect Forms → Generate & Fill
4. AI 将生成中文数据：
   - 中文姓名
   - 中国手机号
   - 中文地址
   - 其他中文信息

### 示例 3：测试多个表单

1. 打开包含多个表单的页面
2. 扩展会检测所有表单
3. 在预览中可以看到每个表单的字段
4. 点击 Generate & Fill 一次性填充所有表单

## 🔧 高级功能

### 使用自定义 AI API

如果您有自己的 AI 服务或其他兼容 OpenAI API 的服务：

1. 在 Settings 中修改 **API URL**
2. 例如：
   - 阿里云百炼: `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`（默认）
   - Azure OpenAI: `https://your-resource.openai.azure.com/openai/deployments/your-deployment/chat/completions?api-version=2023-05-15`
   - 本地服务: `http://localhost:8000/v1/chat/completions`

### 调试技巧

**扩展不工作？**
- 刷新页面重试
- 确保页面完全加载
- 检查浏览器控制台是否有错误

**查看日志：**
- 右键点击网页 → 检查 → Console
- 搜索 "AI Form Filler" 相关日志

**API 调用失败？**
- 检查 API 密钥是否正确
- 确认账户有足够的余额
- 检查网络连接
- 确保使用正确的百炼 API 地址

## ⚠️ 注意事项

1. **API 费用**：使用阿里云百炼 API 会产生费用，请关注您的使用情况
   - Qwen-Turbo: 约 ¥0.008/千 tokens
   - Qwen-Plus: 约 ¥0.04/千 tokens
   - Qwen-Max: 约 ¥0.12/千 tokens

2. **隐私安全**：
   - API 密钥仅存储在本地
   - 不会发送表单的实际数据给第三方
   - 只发送字段元数据（名称、类型等）给 AI

3. **使用限制**：
   - 某些网站可能有反自动化措施
   - 动态生成的表单可能需要重新检测
   - iframe 中的表单暂不支持

## 🆘 常见问题

### Q: 点击 Detect Forms 没有反应？
A: 确保页面已完全加载，尝试刷新页面后重试。

### Q: AI 生成的数据不准确？
A: 
- 尝试切换到 Qwen-Max 模型
- 确保字段的 name/label 属性清晰明确
- 检查是否选择了正确的语言

### Q: 某些字段没有被填充？
A: 
- 某些网站使用自定义表单组件
- 字段可能是动态生成的
- 尝试手动触发页面重新渲染

### Q: 如何更改生成的数据？
A: 
- 点击 Generate & Fill 会重新生成
- 每次生成都是随机的
- 可以修改 AI 模型影响生成质量（Qwen-Max 质量最好）
